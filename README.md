# Prompt Engineering Showcase

This repository contains Task 1 for the VedGrow Generative AI Internship.

## Task Objective

Document and demonstrate 20+ effective prompts across different use cases with analysis.

## Requirements Covered

- Zero-shot prompts
- Few-shot prompts
- Chain-of-thought style prompts using safe wording such as brief reasoning summaries and clear steps
- 5 domains covered:
  - Coding
  - Writing
  - Summarization
  - Analysis
  - Creative
- Bad prompt vs optimized prompt comparison
- Reasoning behind every improvement
- Reusable prompt template library

## How to Use

Open `Prompt_Engineering_Showcase.md` or `Prompt_Engineering_Showcase.pdf` to view the full documented prompt library.

The file `Prompt_Template_Library.md` contains reusable templates that can be adapted for future projects.

## Optional: Regenerate PDF

Install the required package:

```bash
pip install -r requirements.txt
```

Run:

```bash
python generate_pdf.py
```
